How to run the programm


1. 3dots -> Terminal -> New Terminal
2. cd typescript
3. npm i typescript
4.Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
5. cls          [screen will be cleared dont worry]
6. tsc calculator.ts
7. node calculator.js
